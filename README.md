# AI-Based Intraday Market Decision Support System

This is an academic project focused on building a web-based
decision support system for intraday stock market analysis.

## Project Status
- Documentation completed
- Core indicator logic implemented
- Streamlit dashboard in progress
- Pattern detection and alerts under development

## Technologies Used
- Python
- Streamlit
- Pandas, NumPy
- Plotly

## Note
This project is currently under development and will be updated regularly.
